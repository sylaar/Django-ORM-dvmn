# Пульт охраны банка

Внутренний сайт для сотрудников банка "Сияние". Если Вы попали в этот репозиторий случайно, то вы не сможете его запустить, т.к. у Вас нет доступа к БД, но можете свободно использовать код верстки или посмотреть, как реализованы запросы к БД.

### Как установить

Необходимо создать переменные окружения:  
`DB_SECRET_KEY` - секретный ключ сайта  
`DB_USER`- имя пользователя БД  
`DB_PASSWORD` - пароль пользователя БД  

Python3 должен быть уже установлен. 
Затем используйте `pip` (или `pip3`, есть конфликт с Python2) для установки зависимостей:
```
pip install -r requirements.txt
```

### Цель проекта

Код написан в образовательных целях на онлайн-курсе для веб-разработчиков [dvmn.org](https://dvmn.org/).

### Пример запуска

```
python main.py
```
